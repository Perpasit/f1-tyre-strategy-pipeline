# 📊 F1 Tyre Strategy Dashboard

This dashboard is built using Streamlit in Snowflake and visualizes
analytics from the Gold layer.

------------------------------------------------------------------------

## Overview

-   Tyre performance across compounds
-   Driver consistency
-   Race-level insights

------------------------------------------------------------------------

## Data Source

-   GOLD_F1_TYRE_STRATEGY
-   GOLD_F1_DRIVER_PERFORMANCE
-   GOLD_F1_RACE_SUMMARY
